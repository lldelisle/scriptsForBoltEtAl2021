Unfortunately, we cannot provide the code which has been used 
for the MCMC approach as it is the subject of a manuscript in preparation.
If the manuscript is not published by the time of acceptation of 
Bolt et al. 2021, it will be released here.

In the figure S3e, for each gene, each cluster, each genotype, it has been run:
for nnormx in 1 2 3 4; do
  title=${nnormx}gauss_${geno}_cl${cluster}_${gene}
  python NAME_1d.py \
    --input meta_data_scRNAseq.txt --metadata1ColName genotype --metadata1Values ${geno} --metadata2ColName seurat_clusters \
    --metadata2Values ${cluster} --lxmax ${lxmax} --output ${title} --geneColName ${gene} --nnormx ${nnormx} \
    --title ${title} --figure ${title}.png --nsampMCMC 1000000 --logevidence ${title}_logevid.txt --nis 10000 &> ${title}.log &
done
In figure 3f, for each genex, each cluster, each genotype, it has been run:
for nnormxy in 1 2 3 4 5; do
  title=${nnormxy}gauss_${geno}_cl${cluster}_${genex}VSHoxd13
  python NAME_2d.py \
    --input meta_data_scRNAseq.txt --metadata1ColName genotype --metadata1Values ${geno} \
    --metadata2ColName seurat_clusters --metadata2Values ${cluster} --lxmax ${lxmax} \
    --lymax ${lymax} --output ${title} --geneXColName ${genex} \
    --geneYColName Hoxd13 --nnormxy ${nnormxy}\
    --title ${title} --figure ${title}.png --nsampMCMC 1000000 \
    --logevidence ${title}_logevid.txt --nis 10000
done
The post processing is described in the R script 4_MCMC_1d_plots.R and 5_MCMC_2d_plots.R
